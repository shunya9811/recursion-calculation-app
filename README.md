# Calculation App

コンピュータサイエンス学習プラットフォーム Recursion のvueコースの課題として作成したプロジェクト「Calculation App」です

## プロジェクトの概要

簡単な電卓機能を持ったアプリケーションです。
四則演算、税率計算、税率設定、符号チェンジが実装されています。

注意点として、小数点計算は未実装です
